## 🧰 Tech Stack
<div style={{ fontSize: '0.8rem' }}>

| 분야 | 기술 및 구현 가능 수준 |
| --- | --- |
| **Language** | **Java (Spring Boot 기반 서비스 개발 가능)** — REST API 설계, 트랜잭션 단위 서비스 구현, JPA 기반 ORM 최적화 가능 |
| **Framework** | **Spring Boot (3.x)** — 계층형 아키텍처 설계, AOP / JWT / ExceptionHandler 직접 구현 가능 |
| **Database**  | **MySQL / Redis** — 복합 인덱스 설계, N+1 문제 해결, 캐싱 및 분산 락(`SETNX`)을 통한 동시성 제어 구현 가능 |
| **Messaging** | **Kafka** — 이벤트 기반 비동기 메시징, Producer-Consumer 토픽 구조 설계, 멱등 처리(Idempotency) 구현 가능 |
| **CI/CD & Infra** | **Docker / Jenkins / Nginx** — 멀티 스테이지 빌드, 환경 변수 관리, GitHub PR 머지 트리거 기반 자동 배포 파이프라인 구축 가능 |
| **Cloud / Server** | **AWS / Oracle Cloud** — EC2 및 Free Tier 환경에서 컨테이너 기반 서비스 배포 및 모니터링 환경 구축 가능 |
| **Monitoring** | **Grafana / Prometheus** — 시스템 리소스 모니터링 및 REST API 성능 추적 대시보드 구성 가능 |
| **Frontend** | **React / Vue** — API 연동 및 상태관리(zustand, Vuex) 기반의 기본 UI/UX 구현 가능 |
| **Collaboration** | **Git / Jira / Notion / Swagger** — Git Flow 전략 기반 브랜치 협업 및 Swagger를 통한 API 문서화 가능 |
---
